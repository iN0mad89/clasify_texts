# Алгоритм класифікації

1. Перевірка чинності документа. Якщо виявлено слова "втратив чинність", "скасовано" тощо — категорія OBS.
2. Якщо знайдено фразу "нечинна однорідність" — також OBS.
3. Якщо знайдено будь-який тригер з категорії Бюджет — результат BUD.
4. Інакше, якщо тригер Науковець — SCI.
5. Інакше, якщо тригер Інститут — INS.
6. Інакше, якщо тригер Регулювання — REG.
7. Інакше, якщо тригер Статус — STA.
8. Інакше, якщо тригер базис ННТД — BAS.
9. Якщо жоден з тригерів не спрацював, але знайдено омоніми — HOM, інакше UNLABELLED.
